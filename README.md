Opposites Explorer 
Opposites Explorer is an interactive educational tool designed to support language development in children with autism. It focuses on teaching antonyms through the use of kinetic typography and multi-sensory feedback.

Key Features
Kinetic Typography: Visual animations that represent word meanings, such as shivering for "Cold" and fire effects for "Fast".

Dual Learning Modes: Includes a Practice Mode for free exploration and a Quiz Mode for structured assessment.

Accessibility: A Dark Mode toggle is provided to assist users with light sensitivity.

Audio Integration: Uses the Web Speech API to narrate words and provide verbal reinforcement.

Technology Stack
ReactJS: For building a reactive and component-based user interface.

JavaScript: For managing application logic and state flow.

CSS3: For grid layouts and keyframe-based animations.

Project Setup
Open the directory containing index.html, app.js, and style.css.

Ensure an active internet connection to fetch React libraries via CDN.

Launch the application using a local server, such as the Live Server extension in VS Code.
